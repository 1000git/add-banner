# Add Banner Plugin
WordPressにて、バナーを表示させるためのプラグイン。
どうもウィジェットやらテーマのカスタマイズやらが煩雑でクライアントに説明めんどくさいので、そんなときにときに重宝するはず。

## [出力方法]
<?php add_banner(1); ?> // サイドバー
<?php add_banner(2); ?> // SPメニュー
※ 135行目あたりにて編集・追加すれば別のバナーリストももちろん作成可能。
